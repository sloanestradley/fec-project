Design System

A living reference for all tokens, type, spacing, and components in use across FECLedger. Everything rendered live — no screenshots. Update this page when components are added or removed.

Token Naming Strategy

Tokens are organized in three tiers. Tier 1 primitives are raw hex values — not CSS variables, just the reference palette. Tier 2 semantic tokens are CSS variables defined in styles.css :root — named by meaning, not appearance. Tier 3 component tokens (not yet built) would be scoped to specific components, like --tag-dem-bg.

border-radius:0 is intentional — form controls are deliberately square. Native <select> (.form-select) is now limited to three contexts: .cycle-select in the tab bar (flush, borderless — always inside .tabs-bar), mobile fallback inside .combo-wrap and .state-combo (hidden at desktop; shown at ≤860px when the custom trigger/dropdown are hidden), and the terminated .toggle-switch on committees.html. Filter dropdowns on browse pages use the custom combo pattern — not native selects. .form-select.wide (min-width:200px) is a local extension on committees.html — stays inline there, not promoted. Missing states (hover, error/invalid) and size variants are deferred to a future design decision. initComboDropdown modes: filterable: true — uses a text <input> as the trigger (.state-combo pattern); opens on focus, filters options by typing, trigger displays the selected state code. filterable: false — uses a <button> as the trigger (.combo-wrap + .combo-trigger pattern); opens on click, trigger text updates to the selected label. Both modes share the same .typeahead-dropdown listbox, ARIA semantics (combobox/listbox/option roles), keyboard navigation (arrow keys, Enter, Escape), outside-click dismissal, and mobile native-select fallback. Page-scoped styles not part of this system: .placeholder (dashed border empty state, used on race.html when no race data is available — stays local to that page). .search-bar-wrap and .search-bar (search.html page-level layout wrappers — defined in styles.css but intentionally page-scoped, not promoted to the shared component system).
